# Frontend - Trivia App

## Getting Setup

> _tip_: this frontend is designed to work with the [Flask-based Backend](../backend) so it will not load successfully if the backend is not working or not connected. Stand up the backend first before running the frontend.

### Installing Dependencies

**Node 18+** is required. Install Node and NPM from [https://nodejs.org/en/download](https://nodejs.org/en/download/).

Install project dependencies from the `frontend` directory:

```bash
npm install
```

### Running the Frontend

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser. The page will reload if you make edits.
