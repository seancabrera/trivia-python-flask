# Trivia App

## Project Overview

This project is a trivia app built using Python, Flask, SQLAlchemy, and PostgreSQL. Udacity provided the React frontend and I implemented the REST API backend, including all endpoints, error handling, and tests.

The app allows users to browse questions by category, search for questions, add and delete questions, and play a trivia quiz with randomized questions.
